# emLightWrap
A gizmo that makes the Light Wrap effect. Based on the emGlow gizmo. You can control how many samples (max 11) you want to use, anamorph deform, intensity, tint, size, falloff, tolerance with clamping, and highlights clamping.
![emLightWrap_gizmo](https://github.com/user-attachments/assets/6d37eb08-0634-4231-ab9c-7b2ae550a2c3)
A mask input allows you to connect your specific mask only to affect an area in your image. 'The Overscan' feature is available with the crop for more control in your bounding box and a checkbox for cropping the image.
